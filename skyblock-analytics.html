A walkthrough of every Hypixel Skyblock API endpoint used in this dashboard. Use these in your own Python, JS, or R projects.

ENDPOINT 1 — BAZAAR SNAPSHOT

GET https://api.hypixel.net/v2/skyblock/bazaar
Returns the full live order book for every Bazaar item. Updates every ~60 seconds. No API key required.

Key fields per item: quick_status.buyPrice, quick_status.sellPrice, quick_status.buyVolume, quick_status.sellVolume, quick_status.buyMovingWeek

What to practice: Bid-ask spread calculation, volume-weighted price analysis, time series if you poll repeatedly, Bertrand market detection.

ENDPOINT 2 — ENDED AUCTIONS

GET https://api.hypixel.net/v2/skyblock/auctions_ended
Returns the last ~150 completed AH sales (refreshes every 60s). No API key required. This is the cleanest dataset — real transaction prices, not just listings.

Key fields: item_name, price, bin (true = BIN sale), timestamp, category

What to practice: Price distribution analysis, BIN vs auction premium, item-level demand curves, Cournot equilibrium detection via price stability.

ENDPOINT 3 — LIVE AUCTIONS (paginated)

GET https://api.hypixel.net/v2/skyblock/auctions?page=0
All active AH listings, paginated (~1000 per page, ~50+ pages). Requires iteration. No API key required for basic access.

Key fields: item_name, starting_bid, highest_bid_amount, bin, end, tier

What to practice: Web scraping at scale, supply-side quantity analysis, listing price vs. final price comparison (Cournot quantity theory).

PYTHON STARTER — BAZAAR ANALYSIS

import requests import pandas as pd r = requests.get("https://api.hypixel.net/v2/skyblock/bazaar") data = r.json()["products"] rows = [] for item_id, item in data.items(): qs = item["quick_status"] buy = qs.get("buyPrice", 0) sell = qs.get("sellPrice", 0) spread = ((buy - sell) / sell * 100) if sell > 0 else 0 rows.append({ "item": item_id, "buy_price": buy, "sell_price": sell, "spread_pct": round(spread, 2), "weekly_volume": qs.get("buyMovingWeek", 0), }) df = pd.DataFrame(rows).sort_values("weekly_volume", ascending=False) # Classify market structure def classify(row): if row.spread_pct < 2 and row.weekly_volume > 100000: return "Bertrand" elif row.spread_pct > 10: return "Monopoly-like" else: return "Cournot" df["market_structure"] = df.apply(classify, axis=1) print(df.head(20))
